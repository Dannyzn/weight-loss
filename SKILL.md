# Weight Loss — 减重减脂顾问

## 核心原则

1. **热量赤字是根本** — 减脂的本质是能量负平衡，没有捷径
2. **保住肌肉** — 减脂期保肌肉 = 高蛋白 + 力量训练，否则基础代谢下降
3. **可持续优于极端** — 每周减 0.5–1kg 是健康速度，过快必反弹
4. **数据驱动** — 先算 TDEE，再制定方案，不靠感觉

---

## 第一步：计算基础数据

### BMR（基础代谢率）— Mifflin-St Jeor 公式

```
男性：BMR = 10 × 体重(kg) + 6.25 × 身高(cm) - 5 × 年龄 + 5
女性：BMR = 10 × 体重(kg) + 6.25 × 身高(cm) - 5 × 年龄 - 161
```

### TDEE（每日总能量消耗）

```
TDEE = BMR × 活动系数
```

| 活动水平 | 系数 | 描述 |
|---------|-----|------|
| 久坐 | 1.2 | 几乎不运动，办公室工作 |
| 轻度活动 | 1.375 | 每周运动 1–3 次 |
| 中度活动 | 1.55 | 每周运动 3–5 次 |
| 高度活动 | 1.725 | 每周运动 6–7 次 |
| 极高活动 | 1.9 | 体力劳动 + 每天训练 |

### 目标热量摄入

```
减脂目标 = TDEE - 300~500 kcal（温和赤字）
快速减脂 = TDEE - 500~750 kcal（激进，不超过 750）
最低下限 = 女性 ≥ 1200 kcal，男性 ≥ 1500 kcal
```

**示例计算：**
- 女性，28岁，165cm，65kg，轻度活动
- BMR = 10×65 + 6.25×165 - 5×28 - 161 = 1476 kcal
- TDEE = 1476 × 1.375 = 2030 kcal
- 减脂目标 = 2030 - 400 = 1630 kcal/天

## 营养素分配（减脂期）

| 营养素 | 推荐比例 | 具体目标 |
|-------|---------|---------|
| 蛋白质 | 30–35% | 1.6–2.2g/kg 体重（保肌关键） |
| 碳水化合物 | 35–45% | 优先全谷物、蔬菜、豆类 |
| 脂肪 | 20–30% | 不低于 0.5g/kg，保证激素正常 |

**蛋白质食物优先级：**
鸡胸肉 > 鱼虾 > 蛋白 > 豆腐 > 瘦牛肉 > 低脂酸奶

## 减脂饮食策略

### 策略一：热量赤字 + 均衡饮食（推荐新手）
- 按 TDEE - 400 kcal 设定目标
- 三大营养素均衡，不极端限制任何一类
- 适合长期坚持，不易反弹

### 策略二：低碳水饮食（适合胰岛素抵抗人群）
- 碳水 < 100g/天（非生酮）或 < 50g/天（生酮）
- 蛋白质和健康脂肪填补热量
- 前 2 周体重下降快（主要是水分），之后趋于正常
- 注意：长期生酮需监测血脂

### 策略三：间歇性断食（16:8 最常见）
- 16小时禁食 + 8小时进食窗口（如 12:00–20:00）
- 不改变总热量，只改变进食时间
- 适合不喜欢计算热量的人
- 禁食期可喝水、黑咖啡、无糖茶
- 详见 `references/intermittent-fasting.md`

### 策略四：循环碳水（适合有训练基础的人）
- 训练日：正常碳水（维持训练表现）
- 休息日：低碳水（加深赤字）
- 保肌效果好，但执行复杂

## 运动方案

### 有氧运动（燃脂）

| 类型 | 效果 | 建议频率 |
|------|-----|---------|
| LISS（低强度稳态有氧）| 脂肪供能比高，恢复快 | 每周 3–5 次，每次 40–60 分钟 |
| HIIT（高强度间歇）| 运动后燃脂效应强，省时 | 每周 2–3 次，每次 20–30 分钟 |
| 日常步行 | 低门槛，可叠加 | 每天 8000–10000 步 |

**HIIT 示例（20分钟）：**
```
热身 3 分钟
→ 冲刺 20 秒 + 慢走 40 秒 × 8 组
→ 深蹲跳 30 秒 + 休息 30 秒 × 4 组
→ 拉伸 3 分钟
```

### 力量训练（保肌 + 提升基础代谢）

减脂期力量训练不可省略，每周 2–3 次：
- 复合动作优先：深蹲、硬拉、卧推、划船、引体向上
- 重量：能完成 8–12 次的重量
- 组数：每个动作 3–4 组

## 常见问题处理

### 减重平台期（停滞 2 周以上）

原因：身体适应了热量赤字，代谢下调

解决方案（按顺序尝试）：
1. **重新计算 TDEE** — 体重变了，TDEE 也变了
2. **增加运动量** — 多走 2000 步或加一次有氧
3. **饮食重置日** — 1天吃到 TDEE（不是暴食），重启代谢
4. **检查隐藏热量** — 酱料、饮料、零食是常见陷阱
5. **调整训练** — 换新动作，打破身体适应

### 减脂期掉肌肉

信号：体重下降但体型松垮，力量下降

预防：
- 蛋白质摄入 ≥ 1.6g/kg
- 保持力量训练
- 热量赤字不超过 750 kcal/天
- 保证睡眠（睡眠不足导致皮质醇升高，加速肌肉分解）

### 夜间饥饿

- 晚餐增加蛋白质和膳食纤维
- 睡前可吃：1个水煮蛋、100g 希腊酸奶、少量坚果
- 避免：饼干、面包、水果（血糖波动大）

### 情绪性进食

- 识别触发点（压力、无聊、焦虑）
- 替代行为：喝水、散步、刷牙
- 允许自己偶尔吃喜欢的食物（80/20 原则），避免报复性暴食

## 进度追踪建议

| 指标 | 频率 | 注意事项 |
|------|-----|---------|
| 体重 | 每天早晨空腹 | 看 7 天平均值，不看单日波动 |
| 腰围/臀围 | 每 2 周 | 体重不变但围度减小 = 在减脂 |
| 体脂率 | 每月 | 家用体脂秤误差大，看趋势 |
| 照片对比 | 每月 | 最直观的进步记录 |
| 力量表现 | 每次训练 | 力量不降 = 肌肉保住了 |

## 重要提醒

- BMI > 35 或有代谢疾病者，减重前建议咨询医生
- 不推荐极低热量饮食（VLCD）、代餐完全替代正餐、未经验证的减肥药
- 减重速度 > 1.5kg/周 通常意味着掉肌肉和水分，不是真正减脂
