# 🏃 Weight Loss - 减重减脂专项顾问

AI 驱动的减重减脂专项顾问技能，提供科学的减脂方案、训练计划和饮食策略。

## 📋 功能特性

- ✅ TDEE/BMR 计算公式 + 示例
- ✅ 四种减脂饮食策略（均衡赤字/低碳/间歇断食/循环碳水）
- ✅ 有氧 + 力量训练方案（含 HIIT 示例）
- ✅ 平台期突破方法
- ✅ 防止掉肌肉的策略
- ✅ 夜间饥饿、情绪性进食的处理方法
- ✅ 间歇性断食完整指南（16:8 详细执行、适应期、各方案对比）

## 🚀 安装方式

### 方式一：使用 ClawHub（推荐）

```bash
clawhub install weight-loss
```

### 方式二：手动安装

```bash
cd ~/.openclaw/workspace/skills
git clone https://github.com/Dannyzn/weight-loss.git
```

## 💡 使用方法

安装后，直接向 AI 助手提问即可自动激活此技能：

**触发词示例：**
- "怎么科学减肥？"
- "减脂期间怎么吃？"
- "遇到减肥平台期怎么办？"
- "如何避免减肥掉肌肉？"
- "间歇性断食怎么做？"
- "HIIT 训练计划推荐"
